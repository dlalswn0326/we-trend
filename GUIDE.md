# Wenners - IT/AI 트렌드 공유 플랫폼 사용 가이드

안녕하세요! 요청하신 IT/AI 트렌드 공유 플랫폼 **Wenners**가 완성되었습니다.
이 문서는 개발 지식이 없는 분들도 쉽게 따라 하실 수 있도록 작성된 **사용 설명서**입니다.

## 🚀 시작하기 (실행 방법)

이미 제가 필요한 모든 설치와 설정을 마쳤습니다. 아래 순서대로 실행만 하시면 됩니다.

### 1. 터미널 열기
VS Code 하단의 터미널(Terminal) 창을 확인해주세요.

### 2. 실행 명령어 입력하기
터미널에 아래 명령어를 입력하고 엔터(Enter)를 누르세요.
```bash
npm run dev
```

### 3. 웹사이트 접속하기
명령어를 입력하면 잠시 후 `Ready in ...` 같은 메시지가 뜹니다.
이제 인터넷 브라우저(크롬, 엣지 등)를 켜고 주소창에 아래 주소를 입력해서 들어가세요:
👉 [http://localhost:3000](http://localhost:3000)

### ⚠️ "보안 오류"나 "PSSecurityException"이 뜨나요?
윈도우 보안 설정 때문에 실행이 막힌 경우입니다. 당황하지 마시고 아래 명령어를 대신 입력해보세요:
```bash
cmd /c "npm run dev"
```

## 📱 기능 사용해보기

웹사이트가 열렸다면, 아래 기능들을 하나씩 체험해보세요!

### 1. 최신 뉴스 가져오기 (관리자 기능)
처음에는 게시물이 없을 수 있습니다. 봇에게 뉴스를 가져오라고 시켜봅시다.
브라우저 주소창에 아래 주소를 입력하고 엔터를 치세요:
👉 [http://localhost:3000/api/news/fetch](http://localhost:3000/api/news/fetch)

화면에 `{"message":"Fetched and processed..."}` 같은 글자가 뜨면 성공입니다!
다시 **홈 화면(http://localhost:3000)**으로 돌아가면 최신 뉴스들이 SNS 글처럼 요약되어 올라온 것을 볼 수 있습니다.

### 2. 글 쓰기
- 화면 하단 가운데에 있는 **'+' (네모난 플러스 버튼)**을 눌러보세요.
- 글쓰기 화면이 나옵니다. 자유롭게 글을 쓰고 **'Post'** 버튼을 누르면 내 글이 등록됩니다.

### 3. 좋아요 & 북마크
- 마음에 드는 글의 **하트 아이콘**을 눌러보세요. 숫자가 올라갑니다.
- 나중에 보고 싶은 글은 **책갈피(북마크) 아이콘**을 눌러 저장할 수 있습니다.

## 💡 참고 사항 (설정 관련)

### AI 요약 기능에 대해
현재는 AI가 요약해주는 척하는 '테스트 모드'로 동작합니다.
진짜 AI(OpenAI)가 뉴스를 읽고 요약하게 하려면, 별도의 **API 키**라는 것이 필요합니다.
혹시 API 키를 가지고 계시다면 말씀해주세요! 적용하는 법을 알려드리겠습니다.

### 데이터베이스
데이터는 `dev.db`라는 파일에 안전하게 저장됩니다. 컴퓨터를 껐다 켜도 글은 사라지지 않습니다.
